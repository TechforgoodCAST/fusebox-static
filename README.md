# fusebox-static

## Setup

Prerequisites: [NodeJS](https://nodejs.org/), [Yarn](https://yarnpkg.com/)

1. Clone repo
2. `cd fusebox-static/`
3. `yarn`

## Development

`npm run dev` to run development server.

## Deployment

1. `npm run build` to compile production assets.
2. `surge dist/ fusebox.surge.sh`
